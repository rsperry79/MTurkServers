For this task we are trying to help students world-wide find other sources to the same information by training the computer to know when a passage matches information.

To do this correctly we need to train the computer to not only detect a few different things. First is are both given answers answering the question presented. Next is are both passages on the same subject, but different answers. Finally, is one or both given answers have an answer that does not answer the question.

First, we need you to label these hits with one of the following:
  • SAME – If both answers give an answer to the question but do not give different answers
  • Different – Both answer the question but give different answers.
  • Missing Answer - One or both answers are not an answer to the question.

Then we need you to comment if the question was difficult or you see something that can improve what we are doing. We also need you to mark things like plagiarism, where one passage has a section copied word for word.
